
# 🧠 Mental Health Prediction in Tech

## 📌 Project Overview
This project uses **Classical Machine Learning** to predict whether employees in the tech industry will seek mental health treatment based on workplace culture and personal history.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Scikit-Learn, Pandas, Numpy, Matplotlib, Seaborn
- **Experiment Tracking:** MLflow
- **Model Persistence:** Pickle

## 📊 Key Results
- **Baseline Model:** Logistic Regression (ROC-AUC: 0.89)
- **Optimization:** Hyperparameter Tuning via GridSearchCV
- **Feature Engineering:** PCA (Principal Component Analysis)

## 📁 Files
- `survey.csv`: Real-world dataset from OSMI.
- `mental_health_analysis.ipynb`: Complete documented pipeline.
